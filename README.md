# mall-app

## 基于vue.js和Node.js的商城网站
前端使用vue-cli进行搭建，后台使用Node.js以及express框架搭建，数据库使用mongoDB.未登录状态只能浏览商城，登录之后可以下订单，加入购物车，编辑收货地址等。
